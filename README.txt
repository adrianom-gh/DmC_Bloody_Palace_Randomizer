////////////////////////////////////////////////////
//To install, follow these steps: //
////////////////////////////////////////////////////

1. Run "randomizer.exe"
2. A command prompt (black window) will appear asking you for some user inputs (colored enemies, max enemies per wave, etc.).
3. Answer the questions!
4. Command prompt will disappear and generate a DevilGame.ini file inside of the folder with "randomizer.exe"
5. Copy the generated "DevilGame.ini" into steamapps\common\DmC Devil May Cry\DevilGame\Published\Content\DLCBloodyPalace
6. Done! Have fun!

Keep in mind that this mod is in its early stages and more features/bug fixes will be added in the future.

////////////////////////////////////////////////////
// Backups //
////////////////////////////////////////////////////
If you need to restore BP to its original settings, I provided backups for all files. You can find them in the 
\resources folder.

////////////////////////////////////////////////////
// To disable the timer: //
////////////////////////////////////////////////////
1. Navigate to the /resources/disable_timer folder
2. Copy the BloodyPalace_main.upk file over to the same path as DevilGame.ini 
	steamapps\common\DmC Devil May Cry\DevilGame\Published\Content\DLCBloodyPalace
3. Done!


////////////////////////////////////////////////////
// Health increase mod: //
////////////////////////////////////////////////////
I also added a modification to enemies where all enemies give health after you kill them. Feel free to install
this if you want. The file is in \resources\disable_timer\disable_timer_and_give_more_health

I added this change because I found that BP randomized can be brutal. Adding extra health per kill should make the 
experience more enjoyable. 

////////////////////////////////////////////////////
// Damage colored enemies: //
////////////////////////////////////////////////////
I added SamD's mod that allows users to attack colored enemies with any weapon. You can find this in the 
\resources folder.
